
# 선태의 선택 (Sun-tae's Choice)

## 1단계: 브랜딩 & 레이아웃
- 검은색/노란색 컬러 테마 적용 (김선태 채널 아이덴티티)
- 상단 히어로 배너 (채널명 + 소개 텍스트)
- 반응형 레이아웃 구성

## 2단계: TOP 3 대시보드
- 투표 1, 2, 3위 기업을 큰 카드로 표시
- 순위 변동 애니메이션
- 파란색 강조 구역

## 3단계: 실시간 팩트 보드 (좌측)
- 기업 목록을 좋아요 순으로 정렬하여 표시
- 기업 로고, 채널명, 댓글 원문, 좋아요 수 노출
- 목업 데이터로 우선 구현 (추후 YouTube API 연동)

## 4단계: 유저 예측 투표소 (우측)
- 인증 채널 리스트에서 클릭 한 번으로 투표
- localStorage 기반 1일 1회 중복 투표 방지
- 투표 수 실시간 반영

## 5단계: Supabase 백엔드 연동
- `companies` 테이블: 기업 정보 + 유튜브 데이터 캐싱
- `votes` 테이블: 투표 기록 (IP, user_agent, created_at)
- Supabase Realtime으로 실시간 투표 수 업데이트

## 6단계 (추후): YouTube API 연동
- Edge Function으로 YouTube Data API v3 호출
- 특정 영상의 인증 채널 댓글 자동 수집
- 좋아요 수 주기적 갱신
